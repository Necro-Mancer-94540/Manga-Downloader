How to use:
1. Type "connect" and press enter to establish connection.
2. Type the manga to be searched (atleast 2 characters) and press enter.
3. Choose whether to read the manga or download it.
4. If read, type the index number of chapter and press enter.
5. When reading, type "exit" and press enter to stop reading.
6. If download, after selecting the chapters to be downloaded type "download" and press enter.
7. When downloading, wait for server to extract the pages (1.5 sec/chapter).
8. Download will start automatically after the extraction is completed.
9. Type "disconnect" and press enter to close the connection.
